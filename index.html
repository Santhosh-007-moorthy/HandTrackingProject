About This

Our Body Gesture Tracking Project is designed to detect and monitor human movements using computer vision. It recognizes gestures and actions in real time through your camera using Python, OpenCV, and MediaPipe.

This project focuses on real-time human body gesture recognition using Python, OpenCV, and MediaPipe. It captures video input from a webcam, detects key body landmarks, and analyzes movements such as hand positions, gestures, and postures. By setting specific conditions, the system can identify errors like incorrect gestures, prolonged stillness, or hands moving out of range. Such a system can be useful in motion-based training, surveillance, and rehabilitation monitoring applications.

The Body Gesture Tracking System is an innovative project aimed at understanding and analyzing human gestures through AI-powered computer vision. It uses pose estimation techniques to track hand and body movements with high accuracy. This project demonstrates my skills in Python programming, computer vision, and machine learning concepts while addressing real-world applications like gesture control, sign recognition, and movement monitoring.
